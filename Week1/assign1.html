What's the difference between the class and the id HTML attribute?

Multiple elements can be assigned to one class, while only one element can be assigned to one id and each id can be used once on the page.

What does DOM stand for? Explain with your own words.

DOM stands for Document Object Model, which is the hierachical structure in HTML. This is sorted in parents and children, where body and title for example is a child to html.

What Python structure does the array correspond to?

The array structure corresponds to the python element list.

Which Python container does object correspond to?

The html equivalent of objects are classes.
